## Step 1. Clone the project
```
$ git clone https://github.com/maximhorse314/testing-backend.git
```

## Step 2. Install dependencies
```
$ cd testing-backend
$ npm install
```

## Step 3. Fill in environment variables
```
$ cp .env.example .env

PORT=3001
```

## Step 4. Start the server
```
$ npm run start
```
